Features Implementation
Luxury Theme: Deep midnight blue background with gold accents (#D4AF37) and glassmorphism cards.
Hero Section: High-quality Porsche imagery (generated) with "Get Started" CTA.
Process Steps: "Request", "Negotiate", "Drive" steps with icons and hover effects.
Request Form: Fully validated form capturing Make, Model, Year, Budget, Location, and Contact info.
Note: Currently simulates submission by logging to console after 1.5s delay.
How to Run locally
Navigate to the web directory and start the server:

cd web
bun run dev
Open http://localhost:3000 to view the site.
